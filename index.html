How it Works

Three steps. No training required. No new software to learn.

1

Put Berengario in CC

CC Berengario on any email or forward it a document. It reads attachments, parses the content, and adds everything to your team's knowledge base — silently, in the background.

2

Send it a question

Write Berengario an email like you'd write a colleague. "What's the protocol for X?" or "Where do I find the policy on Y?" — plain language, no special syntax.

3

Get your answer

Berengario replies within a minute with a clear, sourced answer drawn from everything your team has ever shared with it. Like a PA who's read every document.

See it in action

Teaching and querying happen entirely through email — the interface everyone already uses.

Teaching — just CC
From: researcher@lab.ac.uk
To: team@lab.ac.uk
CC: ask@berengar.io
Subject: Updated Western Blot Protocol v3

Hi all, attached is the updated Western Blot protocol with the new antibody dilutions we discussed.

western_blot_v3.pdf
Querying — just email
From: new_postdoc@lab.ac.uk
To: ask@berengar.io
Subject: Western blot antibody dilutions?

Hi, what antibody dilutions should I use for the Western Blot?

From: ask@berengar.io

According to the Western Blot Protocol v3, the recommended antibody dilutions are:

Primary antibody: 1:1000 in 5% BSA/TBST
Secondary antibody: 1:5000 in 5% milk/TBST

Source: western_blot_v3.pdf (updated Feb 2026)

Most AI tools ask you to change the way you work — learn a new interface, write prompts in a chat window, drag files into yet another platform. Berengario doesn't. It lives where your team already works: email.

Teach it like a colleague

Put Berengario in CC and it learns. Forward a document and it reads it. No uploads, no dashboards, no training sessions. If you can send an email, you can teach Berengario.

Ask it like a colleague

It remembers everything

Features

Everything you need to build and query a team knowledge base.

Multi-format documents

PDF, DOCX, TXT, CSV, XLS, XLSX — just attach and send.

Web interface

Chat-style web UI with conversation history and source citations.

Admin panel

Manage documents, monitor queries, view analytics, control whitelists.

Query optimization

LLM-based query rewriting for better retrieval accuracy.

Document enhancement

Automatic narrative expansion and Q&A generation for structured data.

Web crawling

Ingest content from URLs — send a link, Berengario reads the page.

Function calling

Calendar event creation and export formatting via natural language.

Custom prompts

Per-instance system prompts, email footers, and response formats.
